## 📝 提问历史记录要求

当引用本README文件进行提问时，请在"提问历史"侧边栏中记录：

### 记录格式
- **问题摘要**：简洁概括用户提出的问题
- **涉及知识点**：列出相关的专利法知识点
- **解答要点**：核心解答内容（非对话形式）
- **理解建议**：帮助记忆和理解的建议

### 示例
```
问题：外观设计专利保护期限的过渡规定
知识点：专利法第四次修改、外观设计保护期
解答：2021年6月1日前申请为10年，当日及之后为15年
建议：记住关键时间节点2021.6.1
```
