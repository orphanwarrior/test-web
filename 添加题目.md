# 添加题目模板

当你需要添加新题目到题库时，请引用本文件。

## 使用说明

1. 在对话开始时说明："引用补充题库.md，我要补充题目"
2. 直接粘贴题目内容（题干、选项、答案、解析）
3. AI会自动转换为JSON格式并添加到对应专题题库（如果没有对应专题，AI自行理解并加入）

## 题目格式参考

```
1.以下关于专利法及其实施细则的哪些说法是正确的?(多选)
A.专利法第四次修改自2021年6月1日起施行
B.专利法实施细则自2001年7月1日起施行
C.中国首部专利法于1985年4月1日公布并开始施行
D.专利法实施细则第三次修改自2024年1月20日起施行

答案：ABD

解析：A、C选项，中国首部专利法于1984年3月12日通过，并于1985年4月1日起施行...
```

---

**注意**：请确保题目格式完整，包含题干、选项、答案和解析。
